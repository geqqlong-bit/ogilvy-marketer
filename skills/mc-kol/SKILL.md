## 工作方式

1. 确认达人营销范围：
   - **平台**：小红书 / 抖音 / TikTok / Instagram / YouTube / B站 / 多平台
   - **目标**：声量/种草（曝光导向）/ 转化/带货（ROI导向）/ 品牌建设（调性导向）
   - **预算**：总预算 + 期望分配
   - **品类**：决定达人类型和合作方式
2. 读取 `brand.md`（品牌人格和调性 → 达人匹配标准）
3. 读取 `storyteller.md`（叙事角色 → 达人在故事中的定位）
4. 读取 `brief.md`、`insight.md`（如存在）
5. 产出写入 `campaigns/{project-slug}/kol.md`

```
         ┌───────────┐
         │  头部 KOL   │  1-3 位    品牌背书+破圈
         │  >100万粉   │
         ├───────────┤
      ┌──│ 腰部 KOL    │──┐  10-20 位  深度种草+内容沉淀
      │  │ 10-100万粉  │  │
      ├──┼───────────┼──┤
   ┌──│  │ 尾部 KOC    │  │──┐  50-200 位  真实口碑+长尾搜索
   │  │  │ 1-10万粉    │  │  │
   ├──┼──┼───────────┼──┼──┤
┌──│  │  │  素人/用户   │  │  │──┐  500+ 位  社会证明+UGC积累

└──┴──┴──┴───────────┴──┴──┴──┘
```

### 分层预算分配模型

**模型 A：声量优先（新品上市/品牌破圈）**

| 层级 | 预算占比 | 目标 |
|------|---------|------|
| 头部 KOL | 30-40% | 制造话题，带来关注 |
| 腰部 KOL | 30-40% | 深度种草，内容沉淀 |
| 尾部 KOC | 15-20% | 真实体验，口碑铺底 |
| 素人 | 5-10% | UGC 氛围 |

**模型 B：转化优先（带货/ROI驱动）**

| 层级 | 预算占比 | 目标 |
|------|---------|------|
| 头部 KOL（带货型） | 20-30% | 直播间/视频直接转化 |
| 腰部 KOL | 40-50% | 性价比最高的种草转化 |
| 尾部 KOC | 20-30% | 长尾搜索承接 |
| 素人 | 5% | 评论/口碑 |

**模型 C：品牌建设优先（长期调性）**

| 层级 | 预算占比 | 目标 |
|------|---------|------|
| 头部 KOL（调性型） | 40-50% | 品牌联名/共创，提升品牌势能 |
| 腰部 KOL | 30-40% | 品牌内容资产积累 |
| 尾部 KOC | 10-15% | 真实用户证言 |
| 素人 | 5% | 社会证明 |

## 模块 2：达人筛选标准

### 硬指标（可量化）

```markdown
## 达人筛选 — 硬指标

| 指标 | 健康值 | 红线值 | 说明 |
|------|--------|--------|------|
| 粉丝量 | 与层级匹配 | — | 仅用于分层，不作唯一标准 |
| 互动率 | >3%（小红书）/ >2%（抖音）| <0.5% | 粉丝活跃度真实性 |
| 赞粉比 | 3:1 - 10:1 | >30:1（刷量嫌疑） | 内容质量与粉丝匹配度 |
| 评论质量 | 真实讨论为主 | 大量"好"单字评论 | 判断粉丝是否真实 |
| 近 30 天发布频率 | 3-15 条/月 | 0 或 >30（营销号） | 活跃且稳定 |
| 爆文率 | >10% 帖子超平均互动 | 全部平平 | 内容能力 |
| 商业内容占比 | <40% | >70%（广告号） | 用户信任度 |
```

### 软指标（需人工判断）

| 维度 | 评估方式 |
|------|---------|
| **品牌调性匹配** | 达人的人设、审美、生活方式是否和品牌一致 |
| **内容质量** | 文案能力、拍摄能力、故事性 |
| **粉丝画像匹配** | 达人的粉丝 ≈ 品牌的目标用户？（而不是看达人本人的年龄性别） |
| **口碑/争议** | 有没有负面事件、翻车记录、立场争议 |
| **合作态度** | 是否接受 Brief 修改、是否配合排期 |
| **独特性** | 是否有一个清晰的"人设标签"（不是万金油型） |

### 排除清单

以下类型达人建议直接排除：

- 近期有品牌负面争议/翻车
- 粉丝画像与品牌目标人群严重不符
- 商业内容质量明显低于日常内容
- 同品类竞品合作过于频繁（用户已审美疲劳）
- 刷量/买粉痕迹明显
- 在品牌价值观上有公开冲突的立场

## 模块 3：各平台达人策略

### 小红书

**达人生态特点：**
- 图文 + 视频双形态，图文仍有很强的搜索价值
- KOC（1-10万粉）是性价比核心，真实体验感强
- 爆文的关键不是粉丝量，是内容质量和话题切入点
- 搜索长尾效应强：一篇好内容可以持续带流量几个月

**合作形式：**

| 形式 | 适用层级 | 定价参考 | 内容方向 |
|------|---------|---------|---------|
| 图文笔记 | 全层级 | 0.1-0.3 元/粉 | 种草/测评/教程/合集 |
| 视频笔记 | 腰部以上 | 0.2-0.5 元/粉 | 开箱/使用过程/vlog |
| 直播 | 腰部以上 | 坑位费+佣金 | 限时促销/新品首发 |
| 置换合作 | 尾部/素人 | 产品成本 | 真实体验分享 |
| 品牌合作人 | 头部 | 定制报价 | 联名/长期代言 |

### 抖音

**达人生态特点：**
- 纯视频平台，内容生产门槛高
- 算法推荐为主，粉丝关系弱于小红书
- 带货转化路径成熟（抖音小店+直播间）
- 头部达人影响力集中，腰部达人性价比需精算

**合作形式：**

| 形式 | 适用层级 | 内容方向 |
|------|---------|---------|
| 短视频种草 | 全层级 | 产品展示/剧情植入/测评 |
| 直播带货 | 腰部以上 | 转化导向 |
| 品牌挑战赛 | 配合头部达人发起 | UGC 裂变 |
| 短剧植入 | 头部/中头部 | 软性曝光 |

### TikTok

**达人生态特点：**
- 与抖音逻辑类似但生态独立
- Spark Ads 可以将达人内容直接转为信息流广告
- TikTok Shop 达人带货处于快速增长期
- 海外 KOC 的性价比极高（对比传统 Instagram KOL）

### Instagram

**达人生态特点：**
- 视觉审美第一，品牌调性匹配至关重要
- Stories 合作（24小时限时）成本低，适合测试
- Reels 是增长引擎，算法推荐权重高
- Collab Post（联合发布）可以同时出现在品牌和达人主页

### YouTube

**达人生态特点：**
- 长视频深度种草，信任建设最强
- SEO 长尾效应极好（视频搜索排名持续数年）
- 制作成本高，合作报价也高
- Shorts 是新增量，与 TikTok 类似逻辑

### B站

**达人生态特点：**
- 用户对"恰饭"容忍度高（前提是内容好）
- 弹幕互动是独特的社区信号
- 适合需要深度讲解的产品（科技/游戏/知识类）
- 年轻男性用户比例高

## 模块 5：达人合作管理

### 合作全流程

```
达人筛选 → 初步沟通 → Brief发送 → 报价确认
                                        ↓
        数据回收 ← 发布上线 ← 终审确认 ← 初稿审核
                     ↓
              效果评估 → 长期关系维护 / 结束合作
```

### 达人管理表

```markdown
## 达人合作跟踪

| 达人 | 平台 | 粉丝量 | 层级 | 合作形式 | 报价 | Brief状态 | 初稿状态 | 发布状态 | 数据 |
|------|------|--------|------|---------|------|----------|---------|---------|------|
```

### 常见合作风险与应对

| 风险 | 预防 | 应对 |
|------|------|------|
| 达人拖稿/放鸽子 | 合同明确交付时间+违约条款 | 备选达人 Plan B |
| 内容质量不达标 | Brief 详细 + 提供参考内容 | 最多两轮修改，超出重新协商 |
| 发布后数据差 | 不把所有预算给一个达人 | Spark Ads 加推好内容 |
| 达人翻车/负面 | 合作前背景调查 | 评估是否需要撤回/声明 |
| 同质化内容 | Brief 给不同达人不同角度 | 内容差异化编排 |

### 什么内容值得追加投放

- 互动率 > 平台均值 2 倍以上
- 评论中有真实的产品讨论（非水军）
- 内容的搜索长尾表现好（笔记/视频被搜索结果收录）
- 带来了可观测的品牌搜索量增长

## 输出模式

### 快速达人方案

用户说"帮我找几个达人合作"时，输出：
- 推荐分层策略
- 3-5 个达人筛选标准
- 预算分配建议
- 控制在 1 页以内

### 完整 KOL 策略

用户要"系统做一套达人营销方案"时，输出全部 6 个模块。

### 单项执行

用户只需某个模块（"帮我写达人 Brief" / "怎么评估 KOL 效果" / "小红书达人怎么选"），只输出对应模块。

## 交互规则

- KOL 策略可以独立使用，也可以作为 campaign 的一个执行步骤
- 用户可以提供候选达人列表，系统逐一评估
- 用户可以说"预算 10 万，在小红书做护肤品种草"，系统给出完整方案
- Brief 模板可以直接发给达人或 MCN
- 输出完成后告诉用户文件路径，不要在对话中重复全文
