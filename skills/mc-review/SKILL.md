---
name: mc-review
description: 品牌调性 + 合规双审技能。支持中国广告法和海外法规（FTC/ASA/GDPR 等）。读取 campaign 产出或用户提供的文案，执行品牌一致性检查和法规风险扫描，输出通过/阻塞结论和具体修改项。
---

## 触发条件

当用户提到以下任意情形时使用本技能：

- "帮我审一下" / "品牌审校" / "合规检查" / "能不能上线"
- "这段文案有没有问题"
- "检查一下广告法风险" / "FTC compliant?" / "check compliance"
- 提供了一段文案或指向 campaign 目录中的文件

---

## 工作方式

1. 确定审查对象：
   - 如果用户指定了文件路径，读取该文件
   - 如果用户直接贴了文案，审查该文案
   - 如果用户说"审查整个 campaign"，读取 `campaigns/{project-slug}/` 下所有产出文件
2. 读取 `brief.md`（如存在）获取品牌调性、风险边界和表达禁区
3. **自动判断适用法规体系**：
   - 国内平台内容 → 中国广告法 + 平台规则
   - 海外平台内容 → 目标市场法规（FTC/ASA/EU 等）+ 平台规则
   - 跨境 campaign → 两套体系同时审
4. 将审查结果写入 `campaigns/{project-slug}/review.md`

## 品牌审校

### 检查维度

| 维度 | 具体检查项 |
|------|-----------|
| 调性一致 | 全部表达是否符合 brief 中的品牌调性定义 |
| 人群感知 | 目标人群看到后是否觉得"说的是我" |
| 卖点传达 | 核心卖点是否被充分支撑（用细节，不是宣称） |
| CTA 匹配 | 转化动作是否与品牌定调匹配（避免高端品牌用"立即抢购"） |
| 竞品区隔 | 表达是否有足够差异化，还是跟竞品撞脸 |
| 跨平台一致 | 不同平台的内容是否在调性上统一 |
| 本地化适配 | 海外内容是否有文化不适、直译痕迹或语感偏差 |

### 输出格式

```
### 品牌审校

**结论：** 通过 / 需修改

**发现：**
1. [通过/问题] 具体描述
2. ...

**修改建议：**（仅在"需修改"时输出）
1. 原文：{原始表达}
   建议改为：{修改后表达}
   原因：{为什么要改}
```

## 合规审查 — 中国市场

### 广告法高危项

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 绝对化用语 | 高 | "最好/第一/唯一/全网最低" 等 → 违反广告法第9条 |
| 功效承诺 | 高 | "治疗/修复/立刻见效/临床验证" → 非医疗产品禁用 |
| 虚假原价 | 高 | 划线价必须有历史销售凭证 → 违反第28条 |
| 权威背书 | 中 | "专家推荐/医生建议" → 需有真实授权 |
| 数据引用 | 中 | "99%用户好评" → 需有数据来源 |
| 对比贬低 | 中 | 直接点名竞品贬低 → 违反第13条 |

### 国内平台规则

| 平台 | 高频违规项 |
|------|-----------|
| 小红书 | 正文含联系方式（微信号/手机号）→ 限流或删帖 |
| 抖音 | 引导站外交易 → 违规；虚假促销倒计时 → 违规 |
| 微信 | 诱导分享（"转发领红包"）→ 封禁风险 |
| 微博 | 未标注广告标识的商业推广 → 平台处罚 |
| B站 | 恰饭视频未标注"包含商业推广" → 社区负面 |

## 合规审查 — 美国市场 (FTC)

### FTC 核心规则

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 付费披露 | 高 | 所有 sponsored/gifted 内容必须明确标注 #ad / #sponsored / "Paid partnership"，置于内容开头显眼位置。FTC Endorsement Guides 2023 |
| 虚假证言 | 高 | 用户评价必须反映真实典型体验，不能仅展示非典型最佳效果 |
| 功效声明 | 高 | 健康/美容/减重等声明需有"competent and reliable scientific evidence"支撑 |
| 比较广告 | 中 | 可以比较竞品，但对比数据必须准确且可验证 |
| "Free" 声明 | 中 | 标注"免费"时不能隐藏必要条件或额外费用 |
| 退款政策 | 中 | 如果提到退款/保证，必须披露完整条件 |
| 环保声明 | 中 | "eco-friendly/sustainable/green" 需有具体依据，FTC Green Guides |
| 儿童隐私 | 高 | 面向 13 岁以下受众的内容须遵守 COPPA |

### 美国平台规则

| 平台 | 关键规则 |
|------|---------|
| Instagram/Meta | Branded Content Tool 必须开启；Stories 中 #ad 不能被小字/折叠隐藏 |
| TikTok | 必须使用 "Branded Content" toggle；不能在 bio 规避付费披露 |
| YouTube | 必须勾选"包含付费推广"；口播中需在前 30 秒内明确告知 |
| Amazon | 禁止虚假评价和刷单；A+ Content 中不能使用未验证的"best/top/leading"声明 |
| Google Ads | 广告文案不能使用误导性 URL 或虚假倒计时；Special Categories（住房/信贷/就业）受额外限制 |

## 合规审查 — 英国市场 (ASA/CAP)

### 核心规则

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 广告标识 | 高 | 所有广告内容必须标注 #ad，CAP Code 要求"obviously identifiable as a marketing communication" |
| 误导声明 | 高 | 不能误导消费者关于产品性质、效果或价格 (CAP Code 3.1) |
| 功效证据 | 高 | 所有客观声明需有证据支撑 (CAP Code 3.7) |
| 价格声明 | 中 | 必须包含 VAT，促销价需标注原价有效期 |
| 健康声明 | 高 | 食品/保健品健康声明受 EU-retained regulation 管控，需在 NHCR 注册列表内 |
| HFSS 限制 | 高 | 高脂/高糖/高盐食品不能在面向 16 岁以下受众的媒体投放 |

## 合规审查 — 欧盟市场

### GDPR 相关

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 数据收集 | 高 | 落地页/表单收集个人数据需有 opt-in 同意机制（不能预勾选） |
| Cookie 同意 | 高 | 落地页必须有 cookie consent banner，非必要 cookie 不能默认启用 |
| 再营销 | 中 | 使用 pixel/retargeting 需在隐私政策中披露 |
| 用户画像 | 中 | 基于用户行为的个性化广告需在隐私政策中说明 |
| 数据传输 | 高 | 向 EU 以外传输数据需有合法机制（SCCs/adequacy decision） |

### EU 广告规则

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 绿色声明 | 高 | EU Green Claims Directive：环保声明需第三方认证支撑 |
| Digital Services Act | 中 | 平台广告必须标注"广告"且披露广告主和投放逻辑 |
| 比较广告 | 中 | Directive 2006/114/EC：可以比较但必须客观可验证 |
| 特殊品类 | 高 | 酒精/烟草/赌博/药品各有独立限制，需按品类逐项检查 |

## 合规审查 — 日本市场

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 景品表示法 | 高 | 禁止"优良误认表示"（夸大品质）和"有利误认表示"（虚假促销）|
| ステマ規制 | 高 | 2023 年起：所有广告内容必须明确标注「広告」「PR」，不能隐藏赞助关系 |
| 薬機法 | 高 | 化妆品/健康食品不能使用医疗效能表述 |
| 特定商取引法 | 中 | 电商必须标注经营者信息、退换货条件和价格总额 |

## 合规审查 — 韩国市场

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 表示广告法 | 高 | 禁止虚假/夸大表示，违反者最高 2 年以下有期徒刑 |
| 推荐保证法 | 高 | 有偿推荐必须标注「유료광고」 |
| 化妆品法 | 高 | 功能性化妆品声明需食药处认证 |
| 电子商务法 | 中 | 必须提供 7 天无条件退货，广告中不能回避此义务 |

## 合规审查 — 东南亚市场

| 市场 | 关键规则 |
|------|---------|
| 新加坡 | ASAS (Advertising Standards Authority of Singapore) Code；广告须真实不误导；医疗/美容广告需 HSA 批准 |
| 印尼 | 广告须使用印尼语；清真产品需 BPJPH 认证标识；化妆品需 BPOM 注册 |
| 泰国 | 广告不得使用国王/王室形象；食品/化妆品广告需 FDA Thailand 事先审批 |
| 越南 | 广告法 2012：价格必须含 VAT；比较广告需有第三方检测数据 |
| 马来西亚 | ASAM Code；清真声明需 JAKIM 认证；不能使用"halal"除非有官方认证 |
| 菲律宾 | Ad Standards Council (ASC) 审查；烟酒广告时间段限制 |

## 敏感品类额外检查（全球通用）

| 品类 | 国内规则 | 海外通用规则 |
|------|---------|-------------|
| 食品 | 不能暗示疗效、不能替代药品 | 营养声明需符合目标市场法规（FDA/EFSA/FSA）|
| 美妆 | 不能使用医疗术语 | 不能声称"治疗"或"治愈"皮肤疾病 |
| 保健品 | 必须标注"本品不能替代药物" | FDA: "This statement has not been evaluated by the FDA" disclaimer |
| 金融 | 必须提示风险、不能承诺收益 | FCA (UK): risk warnings mandatory；SEC (US): past performance disclaimer |
| 教育 | 不能承诺效果、不能贩卖焦虑 | FTC: 就业/收入声明需有数据支撑 |
| 酒精 | 不能面向未成年人 | 各国年龄限制不同（US 21+ / UK 18+ / Japan 20+）；部分国家平台广告全面禁止 |
| 儿童产品 | 未成年人保护法相关规定 | COPPA (US) / UK Age Appropriate Design Code / EU GDPR Art. 8 |

## 门禁判定

审查完成后，在文件顶部写入门禁状态：

```markdown
# 审查报告

**项目：** {项目名}
**审查时间：** {当前日期}
**审查范围：** {审查了哪些文件}
**适用法规体系：** {中国广告法 / FTC / ASA / EU / 多市场}

## 门禁状态

| 门禁 | 状态 | 说明 |
|------|------|------|
| 品牌审校 | ✅ 通过 / 🚫 需修改 | {一句话概括} |
| 合规审查（{市场}） | ✅ 通过 / 🚫 需修改 | {一句话概括} |
| **综合判定** | **✅ 可上线 / 🚫 未达上线标准** | |

> 所有生成内容仅供参考，上线前请由具备资质的法务或合规人员复核。
> 海外市场合规审查不构成法律意见，建议在目标市场聘请当地法律顾问确认。
```

## 交互规则

- 用户可以只做品牌审校（"只看品牌调性"）或只做合规审查（"只查合规"）
- 用户可以指定市场（"按美国 FTC 标准审"/ "按日本景品表示法审"）
- 跨境 campaign 默认同时审国内 + 海外法规，在报告中分章节呈现
- 用户修改文案后可以说"再审一遍"，系统重新审查并更新 review.md
- 审查粒度可调：用户可以说"只审 Instagram 的内容"
- 发现高风险问题时，在对话中直接高亮提醒，不只写在文件里

## 交付规范

执行完成后**必须**以下方格式收尾，完整审核报告不得出现在对话中：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ mc-review · 审核完成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📁 文件：campaigns/{slug}/review.md

📌 执行摘要
   · 门禁结论：✅ 可上线  /  🚫 需修改（{N} 项高风险问题）
   · 品牌风险：{高/中/低} · {核心问题或"无"}
   · 合规风险：{高/中/低} · {最严重法域 + 问题描述或"无"}

➡️  下一步：{✅ 可发布 / 🚫 按修改清单第 {X} 项优先处理}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

高风险项（🔴）无论如何**必须在对话中直接高亮**，不能仅写入文件。
